Sistema Protea

Una anatomia de funciones para observar, revelar, orientar, desarrollar, experimentar y sostener aprendizaje adaptativo.

Que permite ver

Ordena el ecosistema Protea como una ruta de lectura, desarrollo y aprendizaje para organizaciones vivas.

Como funciona

Articula dispositivos complementarios: IAO, DIAGNOS, Training, Lab y Hub, segun el momento adaptativo de cada organizacion.

Que obtiene la organizacion

Una ruta de trabajo coherente, modular y transferible para pasar de senales dispersas a capacidad adaptativa sostenida.

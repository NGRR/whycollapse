# 04 · Navegación y enlaces

## Objetivo

Conectar Home, páginas internas, Hub e Insights con una navegación estática coherente.

## Modificar

- `index.html`
- todas las páginas en `pages/`
- CSS si se requiere.

## Menú principal recomendado

Desktop:

- Inicio
- Enfoque
- Inteligencia Adaptativa
- Sistema Protea
- Hub
- Contacto

Mobile:

- mantener tab bar narrativa para secciones del Home;
- offcanvas debe incluir páginas internas.

## Links obligatorios desde Home

Sección 03:
- `pages/inteligencia-adaptativa.html`

Sección 04:
- `pages/metodologia.html`

Sección 05:
- IAO → `pages/iao.html`
- DIAGNOS → `pages/diagnos.html`
- Training → `pages/training.html`
- Lab → `pages/lab.html`
- Hub → `pages/hub.html`

Nav:
- Hub → `pages/hub.html`
- Insights → `pages/insights.html`
- Contacto → `pages/contacto.html`

## Footer recomendado

Incluir links:

- IAO
- DIAGNOS
- Training
- Lab
- Hub
- Observatorio
- Insights
- Contacto

## Criterios

- Todos los links funcionan en GitHub Pages.
- Usar rutas relativas.
- No usar enlaces absolutos innecesarios.
- No romper anclas del Home.

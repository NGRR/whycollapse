## Criterio

La web abre con tesis, no con servicios. El flujo principal es: Triarquía → Inteligencia Adaptativa → IAO → Sistema Protea → Plan Adaptativo → Conversación.

## Dependencias

UIkit 3.21.6 por CDN. No requiere build.
